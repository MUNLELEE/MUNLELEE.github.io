系统总线

2021-03-25
1分钟阅读时长
计算机组成原理

一、总线的基本概念

计算机系统的五大部件之间的互连方式有两种，一种是个部件之间用单独的连线，称为分散连接，另一种是将个部件连到一组公共信息传输线上，称为总线连接。早期IO与存储器交换信息时，都需经过运算器，致使运算器停止运算，严重影响了CPU的工作效率。总线是连接多个部件的信息传输线，是各部件共享的传输介质。 （当多个部件与总线相连时，如果出现两个或两个以上部件同时向总线发送信息，势必导致信号冲突。因此在某一时刻，只允许有一个部件向总线发送信息，而多个部件可以同时从总线上接收相同的信息）

二、总线的分类

2.1 片内总线

芯片内部的总线，远快于片外总线

2.2 系统总线

可以分为三类，数据总线、地址总线和控制总线

2.1.1 数据总线

数据总线用来传输各功能部件之间的数据信息，是双向传输总线，位数与机器字长、存储字长有关 （如果数据总线的宽度位8位，指令字长为16位，那么CPU在取值阶段必须两次访问主存）

2.1.2 地址总线

地址总线主要用来指出数据总线上的源数据或目的数据在主存单元的地址或IO设备的地址。地址线的位数与存储单元的个数有关。

2.1.3 控制总线

通常对任一控制线而言，它的传输是单向的。（常见的控制信号：时钟，复位，总线请求，总线允许，中断请求，中断响应，存储器写，存储器读，IO读，IO写，传输响应）

2.3 通信总线

用于计算机系统之间或计算机系统与其它系统。按传输方式可以分为两种，串行通信 （数据在单条1位宽的传输线上，一位一位的按顺序分是传送） 和并行通信 （数据在多条并行1位宽的传输线上，同时由源传送到目的地）。

三、总线特性及性能标准

3.1 机械特性

机械特性是指总线在机械连接方式上的一些性能（插头插座的使用标准，几何尺寸、形状、引脚等）

3.2 电气特性

指总线的每一根传输线上的信号的传递方向和有效的电平范围。 （通常规定由CPU发出的信号为输出信号，送入CPU的信号为输入信号）

3.3 功能特性

是指总线中每根传输线的功能，如地址总线用来传输地址码等。

3.4 时间特性

是指总线中的任一一根线在什么时间内有效。（一般用可用信号时序图来描述）

四、总线性能指标

4.1 总线的传输周期

指一次总线操作所需的时间（包括申请阶段、寻址阶段、传输阶段和结束阶段），简称总线周期。总线传输周期通常由若干个总线时钟周期组成。

4.2 总线时钟周期

即机器的时钟周期

4.3 总线的工作频率

总线上各种操作的频率，为总线周期的倒数（一秒内传送几次数据），若 总线周期=N个时钟周期，则总线的工作频率=时钟频率/N。

4.4 总线的时钟频率

即机器的时钟频率，为时钟周期的倒数

4.5 总线宽度

总线上同时能够传输的数据位数，通常指数据总线的根数。

4.6 总线带宽

可理解为总线的数据传输率（单位时间内总线上可以传输的数据位数）， 总线带宽=总线工作频率×(总线宽度/8)

4.7 总线复用

总线复用是指一种信号线在不同时间传输不同信息。

4.8 信号线数

地址总线、数据总线和控制总线3种总线数的总和为信号线数。

Note：总线带宽=总线宽度×总线频率

五、总线标准

总线标准，可视为系统与各模块、模块与模块之间的一个互连的标准界面。以下为几种总线标准

5.1 ISA总线

使用独立于CPU的总线时钟，因此CPU可以采用比总线频率更高的时钟。 没有支持总线仲裁的硬件逻辑，因此不能支持多台主设备系统，而且ISA上的所有数据传送必须通过CPU或DMA接口来管理。

5.2 EISA总线

在ISA基础上扩充开放的总线标准，从CPU分离出了总线控制权， 是一种具有智能化的总线，能支持多个总线主控器和突发方式（将在之后的标准解释）的传输

5.3 PCI总线

①高性能。不依附于某个具体处理器的局部总线。支持突发工作方式 （这种方式是指若被传送的数据在主存中连续存放，则在访问此组数据时，只需给出一个数据的地址，占用一个时钟周期，其后每个数据的传送各占一个时钟周期，不必每次给出各个数据的地址，因此可提高传输速率）

②良好的兼容性。与ISA、EISA总线均可兼容。

③支持即插即用，任何扩展卡只要插入系统便可工作。

④支持多主设备能力。 允许任何主设备和从设备（对总线没有控制权的设备）之间实现点到点对等存取
