一、函数与极限

1、常见的等价无穷小

（要注意的是常见的等价无穷小需要在$x\rightarrow0$或整体式子趋向于0****才可以使用）

① $x$~$tanx$~$sinx$~$arcsinx$~$e^x+1$~$arctanx$~$ln(x+1)$~$ln(x+\sqrt{1+x^2})$

② $(1-cosx)$~$x^2\over{2}$

③ $\log_a(1+x)$~$x\over\ln a$

④ $(x-sinx)$~$x^3\over6$~$arcsinx - x$

⑤ $(tanx - x)$~$x^3\over 3$

⑥ $(1+bx)^a - 1$~$abx$

⑦ $(tanx - sinx)$~$x^3\over2$

⑧ $a^x - 1$~$xln a$

⑨ $\sqrt[n]{1+x} - 1$~$x\over n$

⑩ $ln(1+x)$~$x$；$e^x-1$~$x$

2、两个重要极限

① $\lim_{x\rightarrow0}{sinx\over x}$ = 1(变体：)

② $\lim_{x\rightarrow\infty}({1 + {1\over x}})^x$ = e(变体：)

3、关于间断点

通常把间断点分为两类，如果$x_0$是函数$f(x)$的间断点，但左极限$f(x_0^-)$及右极限$f(x_0^+)$都存在，那么$x_0$称为函数$f(x)$的第一类间断点，不是第一类间断点的热河间断点，称为第二类间断点。在第一类间断点中，左右极限相等者为可去间断点，不相等者为跳跃间断点。无穷间断点和振荡间断点为第二类间断点。

4、关于函数连续性

基本初等函数在其定义区间内都是连续的，所谓定义区间，就是包含在定义域内的区间。（如果要求取某个函数在某一点是否连续，那么应该是该点的极限等于该点的函数值）。

5、关于多项相加求极限（极限存在性准则）

①如果分子都同次，分母有两项，但是分母两项不同次（也即分子分母有其一有不同次），那么可以考虑采用夹逼定理，其中大项和小项分别为和式的头一项后最后一项。

②如果分子和分母都同次，那么有下面这个式子成立$\lim_{n\rightarrow\infty}{1\over n}\sum_{i=1}^{n}{f({{i}\over n}})=\int_0^1f(x)dx$

以上两种利用了极限存在准则Ⅰ，夹逼定理

①所求极限为一数列，可以先求出数列的上界或者下界，之后再利用递推公式证明单调性

以上利用了极限存在准则Ⅱ，单调有界数列必有极限。

6、几个比较重要的函数比较关系

①当$x>0$时，$ln(x+1)<x$

②当$x>0$时，$sinx<x$

③当$x\neq0$时$e^x>1+x$

④当$0<x<{\pi\over 2}$时，$sinx<x<tanx$

7、极限的性质应用

当题目中只给出了某一点的极限可以考虑极限的保号性。

8、无穷大量与无界变量的关系

无穷大量必为无界变量，而无界变量不一定是无穷大量。

9、求解极限的想法

①当所求的极限没有什么思路时，可以考虑极限与等价无穷小的什么相似，之后变换成相应的等价无穷小形式再应用化简。

②如果在指数上有变量，可以考虑将其转换为$e^{xlnf(x)}$的形式。再做进一步变换。

10、一些可使用结论

1、$\lim_{n\rightarrow\infty}({a^n+b^n+c^n})^{1\over n}=max{{a, b, c}}$

2、$||a|-|b||\leq|a-b|$

二、导数与微分

1、关于可导和连续的问题

可导一定连续，连续不一定可导，不连续一定不可导

2、反函数的导数

反函数的导数等于直接函数导数的倒数

3、关于洛必达法则

洛必达只适用于未定式（$0\over 0$以及$\infty \over \infty$还有一些其它的未定式如$0*\infty$;$\infty-\infty$;$0^0$;$1^\infty$;$\infty^0$）

3.1转换后使用洛必达法则

①$0\over 0$以及$\infty \over \infty$可以直接使用洛必达法则

②$\infty-\infty$可以通过化简、通分、根式有理化和变量替换等方法转换为第一种类型

③$0*\infty$转化为$0\over 0$或$\infty \over \infty$再计算（$0*\infty$→$0*{1\over 0}$→$0\over 0$、$0*\infty$→${1\over \infty}*\infty$→${\infty\over \infty}$

④其余形式可以利用$limf(x)^{g(x)}=e^{limg(x)lnf(x)}$转化为其它形式再进行计算

4、偶尔用到的结论

①若$f(x)$是奇函数，则$f'(x)$是偶函数

②若$f(x)$是偶函数，则$f'(x)$是奇函数

三、微分中值定理与导数的应用

1、关于零点的问题

零点问题有时候可以采用罗尔定理或者拉格朗日中值定理解决

2、本章的求导问题

对指数函数进行求导的时候可以采用两边取对数的方法，之后只需要解出$y'$即为原函数的导数

3、证明函数存在某点

可以使用连续函数介值定理：设函数$y=f(x)$在闭区间[a, b]上连续，则在这个区间上必有最大最小值$f(min)=A$和$f(max)=B$且$A\neq B$那么不论C是A与B之间的怎样的一个数，在开区间(a, b)内至少有一点$\xi$使得$f(\xi)=C(a<\xi<b)$

四、不定积分

1、原函数存在定理

连续函数一定有原函数

2、正割函数和余割函数

①$secx={1\over cosx};$$1+tan^{2}x=sec^{2}x$

②$cscx={1\over sinx}$;$1+cot^{2}x=csc^{2}x$

3、三角函数的积分

①一般的对于$sin^{2k+1}xcos^{n}x$或$sin^{n}xcos^{2k+1}x$可以变换成$u=cosx$或$u=sinx$求结果

②对于$sin^{2k}xcos^{2l}x$型函数，可以利用$sin^{2}x={1\over 2}(1-cos2x)$或$cos^{2}x={1\over2}(1+cos2x)$化成$cos2x$的多项式

③对于$tan^{n}sec^{2k}x$或$tan^{2k-1}xsec^{n}x$可以利用$u=tanx$或$u=secx$求结果

④如果被积函数含有$\sqrt{a^{2}-x^{2}}$可以做代换$x=asint$如果含有$\sqrt{x^{2}+a^{2}}$可以做代换$x=tant$如果含有$\sqrt{x^{2}-a^{2}}$可以做代换$x=±asect$

七、微分方程

1、可分离变量的微分方程

如果一个一阶微分方程能写成$g(y)dy=f(x)dx$的形式，就是说能把微分方程写成一端只含$y$的函数和$dy$，另一端只含$x$的函数和$dx$，那么原方程就称为可分离变量的微分方程。

2、齐次方程

如果一阶微分方程可以化成$dy\over dx={\varphi({y\over x})}$

3、一阶线性微分方程

方程形如${dy\over dx}+P(x)y=Q(x)$因为它对于未知函数$y$及其导数是一次方程，所以为一阶线性微分方程，通常使用常数变易法解。
